Sapphire Yours - Download page

This is the release 3.1 of Sapphire Yours. All future releases of the game will also be available for download on this page.
The game has been designed to be extendible concerning the graphical and sound artwork and the levels that come with it. To get started, you only need the base package. The extension package just adds extra music to it.
We want to say thanks to Erik Eckstein here, who created the demos and the level-information for all the old Emerald Mine levels.
SAPP31.EXE [2202 KB] Base package

Installation

The installation of each package is simply done starting the self-extracting executeable. The installer will then ask you to which directory to you want to install the game. If you do not want to use the default installation path, make sure that you select the same path for all packages
After successful installation, you should find some new entries in your 'Start'-menu. Just select 'Programs' - 'Sapphire Yours 3.1' - 'Sapphire Yours' to start the game.

MIDI-Support

This version of Sapphire Yours comes with title and background music. To play the music, the game relies on the built-in capabilities of Windows, so you have to make sure that the MIDI-Synthesizer is properly set up and activated. In fact you can simply turn background music on and off by using the Volume Control panel of the system.
Sapphire Yours allows you to select any MIDI-file as title or background music. The internet is full with publicly available music, and therefore we used a little compilation of this music for the game extension pack.
We do not own the copyright of these MIDIs, but since they all have been made publicly available by their authors we think it is OK, to redistribute them here.
