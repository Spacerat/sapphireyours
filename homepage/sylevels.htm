Sapphire Yours - Levels!

Dear customers,
This page is intended as a swapping area for levels you have designed yourself and want to make available to a greater audience. Please send all levels you wish to be placed on this list to my e-mail address (see image at bottom of this page). No further check, rating or editing will be done on the levels you send to us. Nevertheless we will refuse to publish any level file containing offensive content (as text or pictogram).

No checking - no support

Note that all these levels are not checked by us for solveability or correct rating. So please directly contact the autor of the levels in question if you run into any problems (and also to express your gratitude for this work).
As the features of the game increase with every new version, levels designed with a new version will probably not work anymore on an older version of the game. So I highly encourage you to install the newest game package before trying any of these levels.

How to access the level files

The newer versions of the game come with an integrated level editor. Anything you design with it will be stored on your computer in the file directory C:/sapp31/levels/custom/ by default. Just use the windows explorer to move levels files to and from this directory.
