Sapphire Yours - Download

Das ist die Version 3.1 von Sapphire Yours. Alle zukünftigen Versionen werden ebenfalls auf dieser Download-Seite verfügbar sein.
Das Spiel ist so ausgelegt, dass es bezüglich Grafik, Sounds und Levels erweiterbar ist. Für den Anfang brauchen Sie nur das Basispaket. Das Erweiterungspaket fügt zusätzliche Sounds, Grafiken, Levels und Musik hinzu.

Installation

Zur Installation der einzelnen Pakete müssen Sie einfach das entsprechende selbstentpackende Programm starten. Das Installationsprogramm fragt Sie dann nach dem Zielverzeichnis, wo das Spiel installiert werden soll. Wenn Sie nicht das voreingestellte Verzeichnis verwenden wollen, dann geben Sie acht darauf, bei allen Paketen das selbe Verzeichnis anzugeben.
Nach erfolgreicher Installation sollte es einige neue Einträge in Ihrem Startmenü geben. Wählen Sie "Programme' - 'Sapphire Yours 3.1' - 'Sapphire Yours' um das Spiel zu starten.

MIDI-Support

Diese Version von Sapphire Yours enthält Musik für das Hauptmenü und als Hintergrund für die Levels. Um die Musik abzuspielen, benötigt das Spiel die in das Windows-System eingebauten Mechanismen. Vergewissern Sie sich also, ob die Systemeinstellungen für den MIDI-Synthesizer richtig gesetzt sind.
Sapphire Yours erlaubt die Verwendung beliebiger MIDIs als Titel- und Hintergrundmusik. Weil es im Internet jede Menge frei verfügbarer Musik im MIDI-Format gibt, haben wir hier eine kleine Sammlung im Erweiterungspaket zusammengefasst.
Wir haben nicht die Rechte an diesen MIDIs, aber weil ihre Autoren diese Musikstücke frei zugänglich im Internet veröffentlich haben, sollte es kein Problem sein, sie hier nochmal zu verteilen.
