Sapphire Yours - What's New ?

This page is intended to act as a journal for all news related to the game Sapphire Yous and to document the development progress. The entries are written in chronological order from bottom to top so that you can see the newest entries at first glance.

25.8.2001 - Version 3.1 released

Due to the moving of the homepage we were also forced to release a new version of the game (because all links to the old homepage did not work anymore of course). Nevertheless there are also some minor bug fixes and a very improved "Graphite" graphics set. This new artwork has been done by Jürgen Bonhagen and is really cool - so try it out (change the game settings in the main menu).

25.8.2001 - Homepage has moved yet again

Unluckily our new web space provider ceased operation 5 weeks after we moved the Sapphire Yours Homepage to it.
So we had to move again... (sigh)
This new location (members.aon.at/sapphire) should be a little more stable now.

30.6.2001 - Version 3.0 released

Features of the new version:
  • The "Pusher"
  • Additional artwork and soundsets integrated directly into base package
  • As always: New levels

24.6.2001 - New address of the Sapphire Yours Homepage

"XPoint" cancelled its operation on private customers and small bussinesses. Therefore the homepage had to move to this new location.

11.2.2001 - Swapping place for levels

From now on the "Levels!" section of the Sapphire Yours homepage gives all users the possibility to make their self-designed levels available to the public.

16.10.2000 - Version 2.9 released

After many months of programming and development we finally came to a new version with all the promised features und much more:
  • Fully integrated level editor
  • Level category system
  • Mouse support for menu selection
  • New music
  • Additional graphics and sound set
  • VCR mode is now easier to handle
  • Additional game element: elevator
  • Movement counter
  • Customiseable special graphics objects
  • Many, many more levels

Summer 2000 - Version 2.9 in developement

In preparation for a version 3.0 that will also be available for linux, the 2.9 for windows will come with:
An integrated level editor
Enhanced menu handling
Mouse support
and as usual much more levels

15.4.2000 - Version 2.1 released

This new version finally has support for:
Multiple languages (englisch and german by now)
Joystick input
Different screen resolutions
Various text font sizes
Adjustable image sizes

And of course there are many more new levels in the game.

5.3.2000 - New layout of the homepage

20.1.2000 - Development start of 2.1

The next release will be available before 13.4.2000 which is the deadline for a shareware contest running at our main distributor zdnet.com. This release will include support for multiple languages, enhanced sound output, improved graphics and of course much more levels.

17.12.1999 - Version 2.0 released

Just in time for christmas, the newest version of Sapphire Yours comes with a full load of enhancements:

New main menu and level selection
Faster graphic engine and adaptive frame rate
Original Sapphire Yours game music
New game elements: Robots, beetles, quicksand
Christmas artwork
More levels
and much more ...

11.11.1999 - Version 2.0 nearly completed

Most of the programming has now been finished. The main focus now is on improving the artwork and the music. Also some work on the levels and the level editor is still necessary.

September 1999 - Development start of version 2.0

The version 2.0 will be available somewhere around christmas 1999.

Most important features:
Completely re-written graphics engine (faster, more consistent)
Improved main menu
New levels
New game objects (beetles)
Visible VCR tracks
Demo editing capabilities

28.6.1999 - Version 1.1 released

Many new levels
New game objects (cushion, ruby)
Minor changes in the graphics
Enhanced managment of finished/known state of levels
Suspended games

29.4.1999 - Version 1.0 released (the Birthday Version)

First official release

1.3.1999 - New name for the game

Until now, I always used the name "Reinhard's Emerald Mine" as product name for my game, but I have been finally convinced to choose a name that will not bring us in conflict with Kingsoft.
This way "Sapphire Yours" was born.

10.2.1999 - Start of the DirectX version

Until this date, the whole development of the game is done on a DOS PC using TurboPascal. Porting the game logic from Pascal to C and the graphics engine from direct VGA-register programming to DirectX will eat up the next month of development.
Reinhard Grafl
